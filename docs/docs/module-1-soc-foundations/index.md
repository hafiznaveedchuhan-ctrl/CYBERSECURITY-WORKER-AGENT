# Module 1: SOC Foundations

Welcome to the first module of the AI-SOC Textbook. This module covers the foundational concepts of Security Operations Centers (SOC) and their role in modern cybersecurity.

## Learning Objectives

By the end of this module, you will be able to:

1. Explain the purpose and structure of a Security Operations Center
2. Identify key roles and responsibilities in a SOC team
3. Understand the alert triage process and severity classification
4. Describe common SOC tools and technologies
5. Apply basic incident response procedures

## Prerequisites

No prior security operations experience is required. Basic understanding of:
- Computer networking fundamentals
- Operating system concepts
- Common cyber threats

Approximately 4-6 hours to complete all readings and exercises.

# Module 2: Agentic AI for Security

This module introduces the concepts of agentic AI and how autonomous agents can enhance security operations.

## Learning Objectives

By the end of this module, you will be able to:

1. Explain what agentic AI is and how it differs from traditional AI
2. Understand the architecture of AI agents
3. Identify use cases for AI agents in security operations
4. Describe the benefits and risks of autonomous security agents
5. Design agent workflows for common SOC tasks
